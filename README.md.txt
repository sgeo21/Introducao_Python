# Projeto Python
------------------------------------------------------------------------

##  1. Descrição

Este projeto é uma aplicação desenvolvida em **Python**.

Tem como objetivo estudar e aprimorar conhecimentos da linguagem.

O Projeto Python tem como objetivo demonstrar atividades desenvolvidas em aulas, ** na atividade bancopy.py é possivel ** selecionar operações através de menu, realizar saque e depósito através de operações diretas e verificação de extrato.

------------------------------------------------------------------------

## 2. Funcionalidades

-  **Visualização e seleção de operações através de menu:** criação, listagem e personalização de treinos e práticas de movimento.
-  **Selecionar operações:** depositar, sacar e ver extrato.
